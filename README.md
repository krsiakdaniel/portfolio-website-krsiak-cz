## ✏️ Project Description

- Built with **React**, **TypeScript**, **Next.js**.

- Proficient in **UX**, **UI** design, scalable code delivery.

- Writing **E2E** tests with **Playwright**.
- Writing **Unit** tests with **Jest**.
